"Garden Constructor" Feedback

Question Text
The Setup

Consider a garden class that helps manage a garden of flowers (each flower is an instance of a flower class). The garden class stores the flowers in a 4x4 grid (private variable flower ** grid).

The Problem

In the class garden, create a constructor which takes one parameter, an array of flowers (flower *). This constructor must:

  • Allocate memory on the heap for grid to store the flower in a 4x4 grid.
  • Store each flower in one position of the grid (there is no requirement on the position of each flower on the 4x4 grid, except that each flower must be in exactly one spot).

You should assume that the input array will always contain exactly 16 flowers (one for each position). You need only to write the constructor (you do not need to write all of the "big three").

Your code must compile and work without modifying any of the other helper functions defined in the class (however, feel free to call them in your code!).

Question Text
The Setup

Consider a schedule class that stores a set of courses that make up your schedule. The schedule class has two private variables:

  • int count, the current number of courses in the schedule. This variable is the length of the array courses.
  • string *courses, an array of course names.
The Problem

In the schedule class, overload the - (minus) operator to the courses contained in one schedule from the other.

The result must be a schedule that contains only the courses found in the first schedule that are not found on the second schedule.

Your code must compile and work without modifying any of the other helper functions defined in the class (however, feel free to call them in your code!).
